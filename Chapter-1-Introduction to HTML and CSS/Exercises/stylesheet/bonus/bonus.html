HTML Tags Reference

<html>

The root element of an HTML document. All other elements must be descendants of this element.

<!DOCTYPE html>
<html lang="en">
  <head>
    <!-- Metadata goes here -->
  </head>
  <body>
    <!-- Content goes here -->
  </body>
</html>

<head>

Contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.

<head>
  <title>Page Title</title>
  <link rel="stylesheet" href="styles.css">
  <meta charset="UTF-8">
</head>

<body>

Represents the content of an HTML document. There can be only one <body> element in a document.

<body>
  <h1>Main Heading</h1>
  <p>This is a paragraph.</p>
</body>

<h1> to <h6>

Heading elements represent six levels of section headings. <h1> is the highest level and <h6> is the lowest.

<h1>Main Title</h1>
<h2>Section Heading</h2>
<h3>Sub-section Heading</h3>

<p>

Represents a paragraph of text.

<p>This is a paragraph of text. 
It can span multiple lines.</p>

<a>

Creates hyperlinks to web pages, files, email addresses, or any other URL.

<a href="https://example.com" target="_blank">Visit Example</a>

<img>

Embeds an image into the document.

<img src="image.jpg" alt="Description of image" width="300">

<ul> and <li>

Creates an unordered list (bulleted list) with list items.

<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>

<div>

A generic container for flow content that has no semantic meaning.

<div class="container">
  <p>Content inside a div</p>
</div>

<span>

A generic inline container for phrasing content that has no semantic meaning.

<p>This is <span class="highlight">important</span> text.</p>

CSS Properties Reference

color

Sets the color of text content.

p {
  color: #333333;
}

h1 {
  color: royalblue;
}

font-family

Specifies the font for an element.

body {
  font-family: 'Arial', sans-serif;
}

margin

Sets the margin area on all four sides of an element.

.container {
  margin: 20px auto;
}

padding

Sets the padding area on all four sides of an element.

.card {
  padding: 15px;
}

background-color

Sets the background color of an element.

header {
  background-color: #f8f9fa;
}

border

Sets an element's border with width, style, and color.

img {
  border: 1px solid #ddd;
}

display

Specifies the display behavior of an element.

.nav-item {
  display: inline-block;
}

text-align

Sets the horizontal alignment of text content.

h1 {
  text-align: center;
}

Practical Examples

Basic Page Structure

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Page</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <header>
    <h1>Welcome to My Site</h1>
  </header>
  <main>
    <p>This is the main content area.</p>
  </main>
  <footer>
    <p>© 2023 My Site</p>
  </footer>
</body>
</html>

Styled Navigation

<nav class="main-nav">
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Services</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>

<style>
.main-nav {
  background-color: #333;
  padding: 10px;
}

.main-nav ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
}

.main-nav li {
  display: inline-block;
  margin-right: 20px;
}

.main-nav a {
  color: white;
  text-decoration: none;
}

.main-nav a:hover {
  text-decoration: underline;
}
</style>
